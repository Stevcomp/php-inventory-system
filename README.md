# Sistema-de-Gestao-de-Estoque-ConstruTech
Sistema de gestão de estoque desenvolvido em PHP e CSS3. Solução focada em controle ágil e prático de estoque de materiais, apresentando a autenticação de usuários, monitoramento de níveis críticos via alertas visuais, filtragem dinâmica por categoria e cálculo automatizado e patrimônio com interface otimizada e visualmente prática.
